# C1.Java ☕

Projeto desenvolvido em Java com foco em aprendizado da linguagem, lógica de programação e conceitos fundamentais de desenvolvimento orientado a objetos.

## 📚 Sobre o projeto

O **C1.Java** é um repositório criado para estudos e prática com Java, reunindo exercícios, exemplos e implementações voltadas ao desenvolvimento acadêmico e evolução técnica na linguagem.

O objetivo do projeto é fortalecer conhecimentos em:

- Sintaxe Java
- Estruturas de decisão e repetição
- Programação Orientada a Objetos (POO)
- Manipulação de dados
- Organização de código
- Boas práticas de desenvolvimento

---

## 🚀 Tecnologias utilizadas

- Java
- JDK 17+ (recomendado)
- IDE de sua preferência:
  - IntelliJ IDEA
  - Eclipse
  - NetBeans
  - VS Code
